# Parameter summary

This topic summarizes the parameters in OceanBase Database Proxy (ODP).

## Parameters related to configurations

| Parameter | Description |
|---------|--------------|
| [refresh_json_config](200.dynamically-effective/1980.refresh-json-config.md) | Specifies whether to force update the JavaScript Object Notation (JSON) information when the Config Server is refreshed.  |
| [refresh_rslist](200.dynamically-effective/1990.refresh-rslist.md) | Specifies whether to update all RootService lists when the Config Server is refreshed.  |
| [refresh_idc_list](200.dynamically-effective/1970.refresh-idc-list.md) | Specifies whether to update all Internet Data Center (IDC) lists when the Config Server is refreshed.  |
