# ip_listen_mode

`ip_listen_mode` specifies the listening mode of OceanBase Database Proxy (ODP).

| Attribute | Description |
|----------|---------|
| Parameter type | Integer |
| Default value | 1 |
| Value range | <ul><li>`1`: indicates the IPv4 listening mode.</li><li>`2`: indicates the IPv6 listening mode.</li><li>`3`: indicates the IPv4 and IPv6 listening mode.</li></ul> |
