# flow_low_water_mark

`flow_low_water_mark` specifies the low watermark for stopping throttling. If you specify a negative value, OceanBase Database Proxy (ODP) still takes it as `64K`.

| Attribute | Description |
|----------|---------|
| Parameter type | Capacity |
| Default value | 64K |
| Value range | [0, 16MB] |
