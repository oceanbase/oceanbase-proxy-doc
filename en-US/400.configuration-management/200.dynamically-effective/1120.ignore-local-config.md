# ignore_local_config

`ignore_local_config` specifies whether to ignore all local cache files and start OceanBase Database Proxy (ODP) by using a remote JSON file.

| Attribute | Description |
|----------|---------|
| Parameter type | Boolean |
| Default value | True |
| Value range | <ul><li>`True`: specifies to start ODP by using a remote JSON file.</li><li>`False`: specifies to start ODP by using a local cache file.</li></ul> |
