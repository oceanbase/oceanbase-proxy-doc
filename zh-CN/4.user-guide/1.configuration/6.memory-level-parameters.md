# 内存级参数

本文介绍 ODP 的内存级参数。

|          **参数**           | **默认值** |             **说明**              |
|---------------------------|---------|---------------------------------|
| **`refresh_json_config`** | false   | 重新从 config server 获取 json 配置。   |
| **`refresh_rslist`**      | false   | 重新从 config server 获取 rslist 配置。 |
| **`refresh_idc_list`**    | false   | 重新从 config server 获取 idc 配置。    |
| **`refresh_config`**      | false   | 重新从 MetaDB 更新所有配置。              |
