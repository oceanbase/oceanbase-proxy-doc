# refresh_config

refresh_config 用于控制是否在表处理器进行检查工作时更新所有代理配置。
<!-- 这里的代理配置是说更新 ODP 的所有配置，还是啥 -->
<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项为内存级配置项。</p>
</main>

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   布尔类型      |
| 默认值   | False     |
| 取值范围 | <ul><li>True：更新所有代理配置</li><li>False：不更新 ODP 配置</li></ul>  |
