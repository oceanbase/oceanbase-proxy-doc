# refresh_json_config

refresh_json_config 用于控制是否在刷新 config server 时强制更新 JSON 信息。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项为内存级配置项。</p>
</main>

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   布尔类型      |
| 默认值   | False     |
| 取值范围 | <ul><li>True：更新 JSON 信息</li><li>False：不更新 JSON 信息</li></ul>  |
