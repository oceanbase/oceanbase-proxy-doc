# enable_abort_conn_info

`enable_abort_conn_info` 用于控制 ODP 是否会在终止连接时发送错误数据包。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项自 4.2.0 版本开始引入，并从 4.2.3 版本开始已删除。</p>
</main>

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   布尔类型      |
| 默认值   | False     |
| 取值范围 | <ul><li>True：发送</li><li>False：不发送</li></ul>  |
