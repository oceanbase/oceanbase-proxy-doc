# enable_qa_mode

`enable_qa_mode` 用于控制 ODP 是否可以强制过期所有位置缓存。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项仅用于测试环境，一般不建议使用。</p>
</main>

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   布尔类型      |
| 默认值   | False     |
| 取值范围 | <ul><li>True：ODP 可以强制过期所有位置缓存</li><li>False：ODP 无法强制过期所有位置缓存</li></ul>  |
