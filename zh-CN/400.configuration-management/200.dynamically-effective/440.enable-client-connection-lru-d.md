# enable_client_connection_lru_disconnect

`enable_client_connection_lru_disconnect` 用于判断客户端连接数达到上限时是否接受新的连接。

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   布尔类型      |
| 默认值   | False     |
| 取值范围 | <ul><li>True：新的连接将被接受，并断开最近最少使用的客户端连接。</li><li>False：新的连接将被拒绝，并返回错误信息数据包。</li></ul>  |
