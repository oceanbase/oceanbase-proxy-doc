# refresh_rslist

`refresh_rslist` 用于控制是否在刷新 Config Server 时更新所有 RS 列表。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项为内存级配置项。</p>
</main>

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   布尔类型      |
| 默认值   | False     |
| 取值范围 | <ul><li>True：更新所有 RS 列表</li><li>False：不更新 RS 列表</li></ul>  |
