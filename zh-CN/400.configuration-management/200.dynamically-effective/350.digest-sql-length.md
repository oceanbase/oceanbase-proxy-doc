# digest_sql_length

`digest_sql_length` 用于设置 SQL 打印长度，0 表示使用默认的 SQL 长度打印。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>ODP 中 SQL 打印会被到 <code>obproxy_digest.log</code> 文件。</p>
</main>

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   整数类型      |
| 默认值   | 1024     |
| 取值范围 |   |
<!-- 没有取值范围 -->
