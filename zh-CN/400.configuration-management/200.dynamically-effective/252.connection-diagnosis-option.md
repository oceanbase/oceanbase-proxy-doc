# connection_diagnosis_option

`connection_diagnosis_option` 通过 bit 位指定当前连接诊断的能力。

<main id="notice" type='explain'>
   <h4>说明</h4>
   <p>该配置项自 4.2.2 版本开始引入。</p>
</main>

|  属性    | 描述     |
|----------|---------|
| 参数类型 | 整数类型   |
| 默认值   | 3     |
| 取值范围 | [0, 3]  |

该配置项用二进制位表示：

* 比特 0：用于指定是否开启连接诊断日志记录

* 比特 1：用于指定是否在内部错误发生时尝试构建错误包
