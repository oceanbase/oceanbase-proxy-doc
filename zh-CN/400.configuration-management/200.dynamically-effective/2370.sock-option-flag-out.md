# sock_option_flag_out

`sock_option_flag_out` 用于设置 ODP 和 OceanBase 数据库之间 TCP 的参数。

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   整数类型      |
| 默认值   | 3     |
| 取值范围 | [0, +∞)  |

该配置项用二进制位表示：

* 比特 0：为 1 表示启用 no_delay

* 比特 1：为 1 表示启用 keepalive

* 比特 2：为 1 表示启用 linger_on

建议配置为 3，即：比特 0 和比特 1 均为 1，表示同时启用 no_delay 和 keepalive。
