# xflush_log_level

`xflush_log_level` 用于设置 xflush 日志打印级别。

|  属性    | 描述     |
|----------|---------|
| 参数类型 |  字符串类型       |
| 默认值   | INFO     |
| 取值范围 | 由高到低有如下级别：<ul><li>`ERROR`：非预期的严重错误，不能自动恢复。</li><li>`USER_ERR`：用户输入导致的错误。</li><li>`WARN`：警告，可预期且程序可恢复的错误。</li><li>`INFO`：提示。</li><li>`TRACE`：任务级少量调试信息。</li><li>`DEBUG`：详细调试信息。</li></ul>  |
