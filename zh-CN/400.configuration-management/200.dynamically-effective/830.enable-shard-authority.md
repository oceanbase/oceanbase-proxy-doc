# enable_shard_authority

 `enable_shard_authority` 用于控制是否对 sharding 用户进行权限检查。

 <!-- 表格里该配置项的上下配置项都暂不透出，需确认下是否是遗漏了这个配置项，该配置项是否需要对外透出 
 回答：无需透出-->

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   布尔类型      |
| 默认值   | False     |
| 取值范围 | <ul><li>True：对 sharding 用户进行权限检查</li><li>False：不对 sharding 用户进行权限检查</li></ul>  |
