# proxy_id

`proxy_id` 用于设置标识 ODP 的 ID，以保障不同 ODP 生成的 Client Session ID 不会发生冲突。

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   整数类型      |
| 默认值   | 空字符串     |
| 取值范围 | <ul><li>当 `client_session_id_version` 设置为 2 时，取值范围为 \[0, 8191]。</li><li>当 `client_session_id_version` 设置为 1 时，取值范围为 \[0, 255]。</li></ul>  |
