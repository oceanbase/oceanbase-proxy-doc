# enable_compression_protocol

`enable_compression_protocol` 用于控制是否开启压缩协议。

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   布尔类型      |
| 默认值   | True     |
| 取值范围 | <ul><li>True：开启压缩协议</li><li>False：不开启压缩协议</li></ul>  |

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项需要重新建立连接才会生效。</p>
</main>
