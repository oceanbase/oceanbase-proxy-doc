# enable_async_log

`enable_async_log` 用于判断是否开启异步日志。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>使用异步日志记录方式时，繁忙情况下可能会丢失一些日志。</p>
</main>

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   布尔类型      |
| 默认值   | True     |
| 取值范围 | <ul><li>True：开启</li><li>False：不开启</li></ul>  |
