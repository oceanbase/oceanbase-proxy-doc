# enable_binlog_service

`enable_binlog_service` 用于控制 ODP 是否向 OBLogProxy 发送 binlog 请求。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项自 4.1.0 版本开始引入，并于 4.2.2 版本弃用。</p>
</main>

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   布尔类型      |
| 默认值   | False     |
| 取值范围 | <ul><li>True：发送</li><li>False：不发送</li></ul>  |
