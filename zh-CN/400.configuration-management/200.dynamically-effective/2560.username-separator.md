# username_separator

`username_separator` 用于设置客户端连接 ODP 时使用的用户名分隔符。

|  属性    | 描述     |
|----------|---------|
| 参数类型 |  字符串列表类型       |
| 默认值   | `:`；`-`；`.`     |
| 取值范围 | 不涉及  |

通过 ODP 连接 OceanBase 数据库时，用户名的格式需包含集群名、实例名和用户名。格式有如下四种，常用格式为前两种。

* 用户名@租户名#集群名

* 集群名:租户名:用户名

* 集群名-租户名-用户名

* 集群名.租户名.用户名
