# bt_retry_times

`bt_retry_times` 用于设置 BeyondTrust SDK 重试次数。

|  属性    | 描述     |
|----------|---------|
| 参数类型 |   整数类型      |
| 默认值   | 3     |
| 取值范围 | [0, 100]  |
